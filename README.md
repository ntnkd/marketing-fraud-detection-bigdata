# Marketing Campaign Analysis & Click Fraud Detection 

## 1. Problem Statement

Online advertising platforms generate millions of click records daily.
However, a significant portion of these clicks may come from fraudulent sources
such as bots or click farms.

This project aims to build a Big Data system to:
- Analyze marketing campaign performance
- Detect fraudulent click behavior
- Provide business insights to avoid advertising budget waste

---

## 2. Objectives

- Process large-scale click data using Apache Spark
- Perform marketing analytics
- Detect click fraud using Machine Learning (Spark MLlib)
- Generate business-oriented reports

---

## 3. Architecture Overview

TalkingData Dataset  
→ HDFS  
→ Apache Spark (ETL & Analytics)  
→ Feature Engineering  
→ Spark MLlib  
→ Reporting & Visualization  

---

## 4. Technologies

- Hadoop (HDFS)
- Apache Spark
- Spark MLlib
- Python (PySpark)
- Parquet format

---

## 5. Project Scope

This project focuses on offline Big Data processing.
It does NOT include:
- Real-time streaming
- Kafka integration
- Production deployment
